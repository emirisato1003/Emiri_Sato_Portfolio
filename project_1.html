New York Times Article

School Project

Project Overview

For my final class project, I designed a dynamic web page showcasing the latest news stories form the New York Times. Leveraging the power of the NYT API, this site offers real-time updates on various news categories. This project is developed as a single-page application(SPA) that fetches news stories based on user selection without reloading the entire page.

My Contributions

On the frontend development side, I designed the user interface for the site utilizing HTML, CSS, and Bootstrap and ensured the website is responsive and accessible across different devices and screen sizes.
On the backend, I coded JavaScript in order to successfully retrieve data from NYT API, and handled API responses and parsing JSON data to extract relevant information including titles, abstracts, bylines, and categories.
